# Stock Management System

## Overview
A Java-based inventory and sales management tool for computer shops, enabling efficient handling of customers, suppliers, products, and financial reporting.

## Features
- Customer, product, and supplier registration  
- Invoice generation and repair request management  
- Stock updates with GRN support  
- Admin and cashier roles with distinct privileges  
- Profit reporting and analytics  

## Tech Stack
- Java (Swing/JavaFX)  
- MySQL Database  
- NetBeans IDE  
- Jaspersoft Studio for reporting  

## Setup
1. Clone the repository.  
2. Configure MySQL database and update credentials.  
3. Build and run using NetBeans IDE.  

## Security
- Hourly database backups  
- Secure login with role-based access  
- Fault tolerance and quick error recovery
